Que2:"Problem: You are given a string (STR) of length N, you have to print the count of all alphabet.(using maps) Input: abcadeecfb Output: a=2 b=2 c=2 d=1 e=2 f=1" 

Ans:

let myMap = new Map();

let s="abcadeecfb";

for(let i=0;i < s.length;i++)
{
  if(myMap.has(s[i])){
   
    let previousCount=myMap.get(s[i]);
    myMap.set(s[i],previousCount+1);
  }
  else{
    myMap.set(s[i],1);
  }
  }
